# ⚡ SysPulse

A modern desktop system monitoring and hardware diagnostics application built with Python and PySide6.

SysPulse provides real-time insights into system performance, hardware specifications, resource utilization, and overall system health through a clean and intuitive user interface.

---

## 🚀 Features

### 📊 Dashboard

- Real-time CPU usage monitoring
- Real-time memory usage monitoring
- Disk utilization overview
- Battery status and information
- Network activity monitoring

### 🖥️ Hardware Information

- CPU model and specifications
- Core and thread count
- Installed memory information
- Operating system details
- System architecture information

### 📈 Performance Monitoring

- Live CPU performance graphs
- Live memory usage graphs
- Historical performance tracking
- Resource usage visualization

### ❤️ System Health

- Battery health information
- Storage status monitoring
- Hardware health overview

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/SysPulse.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
